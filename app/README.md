# AgroSeva - PS04 Autonomous Farm Irrigation & Fertilization System

A React Native mobile application built with Expo and TypeScript for intelligent farm management.

## Features

- Soil Monitoring (Moisture, pH, NPK)
- Weather Intelligence
- AI-Based Decision Engine
- Deep Reinforcement Learning
- Explainable AI
- Farmer Dashboard
- Offline Capability
- Multilingual Ready
- Resource Usage Tracking
- Crop Image Analysis

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on Android:
```bash
npm run android
```

4. Run on iOS:
```bash
npm run ios
```
